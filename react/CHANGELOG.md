# Changelog

## 0.1.0 (2026-02-01)

### Features

- Initial release
- `<MinkProvider>` context provider for SDK initialization
- `useMink()` — core SDK access hook
- `useMinkBalance()` — stealth balance hook with auto-refresh
- `useMinkAuth()` — authentication state hook
- `useMinkInbox()` — inbox management hook
- `useAutoForward()` — automatic inbox forwarding hook
