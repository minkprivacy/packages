# Changelog

## 0.1.0 (2026-02-01)

### Features

- Initial release
- `MinkSDK` class for managing private Solana transactions
- `cloak()` — deposit SOL/SPL tokens into the privacy pool
- `reveal()` — withdraw from the privacy pool to any address
- `getStealthBalance()` — fetch decrypted stealth balance
- Inbox system for receiving private payments
- V1/V2/V3 encrypted output support
- Browser-compatible ZK proof generation (snarkjs)
- Merkle tree proof fetching and validation
- Viewing key derivation and management
- Devnet and mainnet network configuration
